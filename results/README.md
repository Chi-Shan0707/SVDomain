# Results

This directory contains the compact result bundle used by the cleaned repo.

## What is here

- `summary_metrics.json`
  - machine-readable headline numbers
- `comparison_tables.md`
  - one-page cross-topic summary
- `tables/`
  - small CSV tables used directly by the docs
- `figures/`
  - a few stable, paper-facing figures
- `scans/`
  - compact JSON summaries for low-rank necessity

## Recommended reading order

1. `summary_metrics.json`
2. `comparison_tables.md`
3. the matching files under `tables/` and `figures/`

## Current scope

This is intentionally not the full artifact dump from the original workspace. It keeps only the files that the current docs actually cite.
